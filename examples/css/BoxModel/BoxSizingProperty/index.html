CSS Box-Sizing Property Demonstration

Content-Box (Default)

box-sizing: content-box;

With box-sizing: content-box, the width is applied only to the content. The padding and border are added outside of the width, increasing the total size of the element.

For example, the width is set to 300px, but the actual size is 300px (content) + 20px (padding on both sides) + 10px (border on both sides) = 360px.

Border-Box

box-sizing: border-box;

With box-sizing: border-box, the width includes the content, padding, and border. The total width remains 300px, as padding and border are part of the width calculation.

The content's available space is reduced accordingly, but the total size is exactly 300px.
